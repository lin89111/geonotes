Pour créer la base de données et la connexion à la base de données :

1/ Dans le Terminal, lancer mysql en tant qu'utilisateur root :

	mysql --user=root

(ou path_to_mysql/mysql --user=root)

2/ Exécuter les deux instructions suivantes :

	grant all privileges on *.* to 'geonotes'@'localhost'
	identified by "geonotes" with grant option;

(crée un utilisateur local [login=geonotes; password=geonotes] avec tous les droits)

	create database geonotes;

(crée une base de données geonotes)

3/ Fermer MySQL

	quit;

4/ Exécuter les instructions décrites sur cette page web ou à la fin de ce document :

http://blog.hbis.fr/2010/10/05/glassfish-jdbc_pool/

Note 1 : le driver JDBC de MySQL se trouve dans le répertoire libs/ de ce projet ou peut être téléchargé ici : http://www.mysql.fr/downloads/connector/j/
Note 2 : dans les instructions, remplacer
	mydb	par geonotes
	myuser	par geonotes
	mypwd	par geonotes

Instructions à exécuter :

cd <path_to_glassfish>/
(ex : cd ~/glassfish3/)

glassfish/bin/asadmin stop-domain

cp ~/mysql-connector-java-5.1.13-bin.jar glassfish/domains/domain1/lib
(à condition que mysql-connector-java-5.1.13-bin.jar soit dans ~ !)

glassfish/bin/asadmin start-domain

./bin/asadmin create-jdbc-connection-pool --datasourceclassname com.mysql.jdbc.jdbc2.optional.MysqlDataSource --restype javax.sql.DataSource --property User=geonotes:Password=geonotes:URL=jdbc\\:mysql\\://127.0.0.1/geonotes geonotes_pool
(doit afficher "Command create-jdbc-connection-pool executed successfully.")

./bin/asadmin ping-connection-pool geonotes_pool
(doit afficher "Command ping-connection-pool executed successfully.")

./bin/asadmin create-jdbc-resource --connectionpoolid=geonotes_pool jdbc/geonotes
(doit afficher "Command create-jdbc-resource executed successfully.")
